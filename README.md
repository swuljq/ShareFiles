# 🚀 ShareFiles

一款极致轻量、跨平台的局域网文件共享与传输工具。无需公网服务器，无需复杂配置，双击即可在局域网内高速传输文件和文件夹。

---

## ✨ 特性 (Features)

* **⚡ 极速传输：** 基于局域网点对点（P2P/直连）传输，速度仅受限于你的路由器和网卡带宽。
* **🖥️ 跨平台支持：** 支持 Windows, macOS 和 Linux，提供直观的图形化操作界面。
* **🔒 全隐私安全：** 所有数据均在局域网内部流转，绝不上传任何云端服务器，保障隐私安全。
